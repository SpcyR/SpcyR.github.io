*Install curl and Node (v16.x)*

```
$ sudo apt install -y curl
$ curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
$ sudo apt install -y nodejs
```
![](git/nodejs.png) 

*Install Node-RED locally*

```
$ sudo npm install -g --unsafe-perm node-red
$ node-red
```
